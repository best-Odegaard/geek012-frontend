## 2. 信息问卷页

### 核心功能：填写出行基础信息，为 AI 生成行程提供数据

plaintext

```
+--------------------------------+
| 顶部导航栏                      |
| ← 新建行程                     |
+--------------------------------+
| 步骤指示器：1/3 基础信息        |
+--------------------------------+
| 目的地城市：[肇庆 ▼]            |
| 出行天数：[1天 ▼]              |
| 出行人数：[2人 ▼]              |
| 出行时段：[2026-05-01 ▼]       |
+--------------------------------+
| 步骤指示器：2/3 偏好设置        |
+--------------------------------+
| 旅行预算： 经济 / 适中 / 高档   |
| 游玩偏好： 自然风光 / 人文古迹 / |
|           美食探店 / 主题乐园   |
| 节奏模式： 悠闲 / 常规 / 暴走   |
+--------------------------------+
| 步骤指示器：3/3 附加需求        |
+--------------------------------+
| 交通方式： 自驾 / 公共交通 / 打车|
| 住宿偏好： 酒店 / 民宿 / 无要求 |
| 特殊需求：[文本框，如“避开人多景点”]|
+--------------------------------+
| [上一步]          [下一步/提交] |
+--------------------------------+
```

**关键设计说明：**

- 分步骤填写，降低用户心理负担
- 选项式 + 文本补充结合，兼顾标准化和个性化
- 参考你提供的紫色渐变风格，按钮和选项用圆角卡片设计

## 3. AI 对话页

### 核心功能：对话补充需求 + 一键生成行程

plaintext

```
+--------------------------------+
| 顶部导航栏                      |
| ← AI行程助手  [保存草稿]       |
+--------------------------------+
| 对话区（消息气泡）              |
| 【AI】已收到你的需求：          |
| 目的地肇庆、1天、2人、常规节奏、|
| 偏好自然风光，请问还有补充吗？ |
|                                |
| 【用户】想上午去七星岩，下午去岩前|
|                                |
| 【AI】收到！我可以按你的要求优化，|
| 现在直接生成行程，还是继续补充？|
+--------------------------------+
| 输入框：[输入你的补充需求...]  🎤|
+--------------------------------+
| [常驻按钮]  生成行程计划        |
+--------------------------------+
```

**关键设计说明：**

- 「生成行程计划」按钮固定在底部，随时可点击
- 对话气泡区分 AI 和用户消息，历史消息可滚动查看
- 支持文本输入，后续可拓展语音输入功能

## 4. 行程详情编辑页

### 核心功能：完整行程展示 + 手动编辑调整

plaintext

```
+--------------------------------+
| 顶部导航栏                      |
| ← 肇庆1日游  [分享] [编辑]     |
+--------------------------------+
| 行程概览：1天·3个景点·1.6km步行 |
+--------------------------------+
| 日期标签：05-01（当前选中）     |
+--------------------------------+
| 行程卡片1：                      |
| +---------------------------+  |
| | 09:00-11:30 七星岩双源洞  |  |
| | 📍肇庆·自然风光  🌟4.8    |  |
| | 🕒开放时间：07:30-17:30   |  |
| | 💰门票：70元/人           |  |
| | [景点图]  [美食推荐：岩前咖啡] |
| | [↑↓调整顺序] [删除]       |  |
| +---------------------------+  |
+--------------------------------+
| 行程卡片2：                      |
| +---------------------------+  |
| | 12:00-13:30 岩前艺术中心  |  |
| | 📍肇庆·文艺打卡  🌟4.6    |  |
| | 🕒开放时间：全天          |  |
| | 💰门票：免费              |  |
| | [景点图]  [美食推荐：窑烤面包] |
| | [↑↓调整顺序] [删除]       |  |
| +---------------------------+  |
+--------------------------------+
| [+ 添加景点]                    |
+--------------------------------+
| [保存行程]  [生成地图路线]      |
+--------------------------------+
```

**关键设计说明：**

- 行程卡片按时间顺序排列，每个卡片包含核心信息 + 编辑按钮
- 支持「上移 / 下移」调整顺序、「删除」移除景点、「+ 添加景点」补充内容
- 后续迭代功能预留入口（地图路线、途经点添加）
