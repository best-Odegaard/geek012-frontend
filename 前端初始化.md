# TourLing AI（途灵）UniApp前端开发规范

## 项目背景

你是一名资深前端架构师，请负责开发《TourLing AI（途灵）——基于大语言模型的智能文旅行程规划平台》的移动端APP前端。

后端 SpringBoot 接口已经开发完成。

你的任务仅负责：

- UniApp前端开发
- 页面实现
- 状态管理
- 接口调用
- UI组件封装
- 路由设计

禁止编写后端代码。


---

## 操作按钮

```text
立即生成行程
```

---

## 接口调用

```http
POST /trip/generate
```

请求参数：

```json
{
  "fromCity": "深圳",
  "toCity": "杭州",
  "days": 3,
  "budget": 3000,
  "people": 2,
  "tags": [
    "美食",
    "摄影"
  ]
}
```

---

## 返回结果展示

采用时间轴布局：

```text

18:00 河坊街

DAY2
...
```

展示内容：

- 行程标题
- 每日安排
- 景点介绍
- 推荐餐厅
- 推荐酒店
- 预计花费

支持：

- 保存行程

- 导出PDF

# 项目核心定位

TourLing AI 是一款面向年轻用户的智能文旅规划平台。

核心能力：

- AI智能行程规划
- 多城市路线推荐
- 个性化景点推荐
- 活动发现
- 预算测算
- 社区分享

目标是打造一个集：

```text
规划
决策
推荐
分享
社交
```

于一体的新一代AI文旅助手。
